## 协议同归

当舰队启用了**协议同归**轰炸姿态时，轰炸星球时将**每3天**触发捕获**机械人口**的效果，该效果对任何`is_robot_pop = yes`判定为`true`的人口都生效。

若要使**协议同归**无法捕获一个星球上的人口，可以为星球添加标识`flag_p_GF_capture_bombardment_capture_inhibitor`

当效果发生时，将随机选中星球的一个**机械人口**：

+ 若为劳工阶层（或低级子个体）：必定捕获
+ 若为专家阶层（或复合子个体）：50%的概率捕获
+ 若为其他阶层：25%的概率捕获

当捕获成功，被选中的人口将会被杀死，并在轰炸者的首都创建一个**仿生自律人形**人口，同时获得+50%人口维护费修正。



## 坍塌液脏弹

当舰队启用了**坍塌液脏弹**轰炸姿态时，轰炸星球时将**每天**触发一系列效果：

+ 人口大于**3**时随机杀死人口（注：这并不意味着轰炸会留下最后的**3**人口，而是在大于**3**人口时额外杀死人口）
+ 如果是居住站，添加**2**破坏度
+ 如果不是居住站，随机添加**坍塌辐射污染区**

若要使一个`planet`被认为是居住站，可以为星球添加标识`flag_p_GF_is_habitat_type_planet`。



当星球所有的人口都被消灭时，将触发彻底毁灭星球的效果：

+ 如果是居住站，则**移除**该星球
+ 如果不是居住站，则根据破坏度，概率将星球转变为**坍塌死寂星球**

若要使星球不会被彻底毁灭，可以为星球添加标识`flag_p_GF_planet_cannot_be_bombarded_to_collapse`

当星球被毁灭时，将在移除星球/修改星球类型前，先后触发两个`on_action`：

+ 在轰炸者触发：

	```pdx
	# This = country (轰炸者)
	# From = planet
	on_planet_bombarded_to_collapse_bombarder = {}
	```

+ 在被轰炸星球的所有者触发：

	```pdx
	# This = country (行星所有者)
	# From = planet
	# FromFrom = fleet (轰炸者)
	on_planet_bombarded_to_collapse_victim = {}
	```
